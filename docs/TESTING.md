## Исправленные проблемы:

1. ❌ **Ошибка "Cannot declare class Resend"**
   - **Причина**: Класс Resend в библиотеке объявлен без namespace
   - **Решение**: Убрали `use Resend\Resend;` и используем `\Resend::client()`

2. ❌ **Повторное подключение config.php**
   - **Причина**: Файл подключался несколько раз
   - **Решение**: Добавили защиту через `APP_CONFIG_LOADED`

3. ❌ **Предупреждения о сессии**
   - **Причина**: ini_set вызывался после session_start
   - **Решение**: Переместили настройки до session_start
